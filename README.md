# Heart Disease Risk Analysis Dashboard

## Amount of Data to DB

The heart disease dataset used in this project contains approximately 4,500 records. The data includes attributes such as age category, smoking status, diabetes status, BMI, physical activity, and general health. This dataset was imported into the database and used to generate visualizations in the Tableau dashboard.

## Utilization of Data Filters

Filters were used in the dashboard to improve user interaction and allow deeper analysis of the data. Users can filter the dataset by attributes such as Age Category, Smoking Status, and Diabetes Status. These filters help users focus on specific groups and understand how different factors influence heart disease risk.

## Number of Calculation Fields

This dashboard primarily uses built-in aggregation functions such as COUNT to summarize the dataset. No additional calculated fields were created for this analysis. The visualizations rely on the dataset attributes and Tableau’s automatic aggregations.

## Number of Visualizations / Graphs

The dashboard contains multiple visualizations including:

* Pie Chart showing overall heart disease distribution
* Heatmap showing smoking vs diabetes relationship
* Age category chart showing heart disease distribution by age group
